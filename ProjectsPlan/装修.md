
# 玄关设计
1. 顶天立地大柜。
2. 中间必须有放包，钥匙，安装插座。电源一键关闭（除厨房以外）。中间的高度设计是一个问题。
2. 玄关柜子底下留空， 留低灯。
3. 玄关需要有挂雨伞的。


# 厨房
1. 台面选择石英石。
2. 水龙头
3. 移动式插座。（轨道插座）
4. 厨房柜子做到顶。
5. 切菜上面需要感应灯。
6. 拐角拉篮
7. 洗碗机（大一点，应该能放进锅子）
8. 水槽底下插座（两个）。净水器。

# 浴室
1. 双水龙头面盆。
2. 有没有洗人机。最好躺着带按摩。
3. 深水封地漏
4. 防水层需要安装地漏
5. 选择一体盆。

# 客厅设计
1. 沙发和床的规格确定后可以进行开关插槽的位置选择。
2. 电视墙做成投影（）投影客厅啥放上面 放置一个核心，漏出投影。
